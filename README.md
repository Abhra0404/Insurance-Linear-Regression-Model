# Insurance Linear Regression Model

This project uses the `insurance.csv` dataset to explore and predict medical insurance charges with a linear regression workflow in `insurance.ipynb`.

## What I Did In The Notebook

- Performed exploratory data analysis on the insurance dataset
- Cleaned and preprocessed the data
- Applied feature engineering and feature extraction
- Checked correlations, including Pearson correlation analysis
- Built a linear regression model to predict insurance charges

## Dataset

The dataset includes these features:

- `age`
- `sex`
- `bmi`
- `children`
- `smoker`
- `region`
- `charges`

## Files

- `insurance.ipynb` - notebook with the data analysis and model building steps
- `insurance.csv` - source dataset used in the notebook

## Usage

Open `insurance.ipynb` in Jupyter or VS Code and run the cells from top to bottom.
